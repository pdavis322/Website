INTRODUCTION TO HTML

Creating a Web Page

We'll now start to create our Web page using HTML and a small amount of CSS.

A simple text editor such as Notepad on Windows or TextEdit on MacOS should suffice for our purposes. For a more authentic development experience, I recommend using Notepad++, Brackets, or Atom.

Our first step is to create an HTML file. Open your editor, and create a new file called "NewWebPage.html". The name doesn't matter, but make sure to include the ".html" extension.

Recall that HTML is just the "stuff" on a Web page: text, images, buttons, etc. All of this "stuff" are called elements. We can create elements using tags. Tags are written with angle brackets (< and >.) Most HTML elements, such as paragraphs or headers, require an opening and closing tag. The closing tag is written with a backslash (/) before the name of the element. The actual content is written inside the tags: for example, <title>My Website</title> will give the Web page a title of "My Website."

To start our Web page, we'll use a special type of tag: a doctype declaration.
It's written like this: <DOCTYPE html>
This tells our browser to expect an HTML file. Next, we'll create our HTML tags: <html></html> These tags mark the beginning and end of our website.

Next, we'll add the <head> and <body> tags. The head tag will contain the formatting of our website, like CSS and its title. The body tag will include the actual content of the website. We'll now add a title to this section by creating a <title> tag in it. Add whatever name you want inside the tags. Open the HTML file in a browser, and you'll see the title on your tab.
Our code should now look like this: 

<!DOCTYPE html>
<html>
	<head>
		<title>
			My Website
		<title>
	</head>
	<body>
	
	</body>
</html>

Now we'll start adding actual text to our Web page. Add an <h1> tag to the body section. This will create a header header. Headers range from "h1" to "h6," "h1" having the largest font. Again, make the header display whatever you want. I'll use <h1>My Header</>. Open your Web page in your browser, and you'll see something like this:

That's not very attractive. We'll now add CSS to style our website.
In the head section, add a style tag: <style type="text/css"></style> Using this tag, we can modify the style of our HTML elements. In the style tag, add a h1 modifier like this: 


<style type="text/css">
	h1 {
	    
	}
</style>

Inside these braces ({}), we can add attributes that specify the font, the color, etc. For now, we'll change the font to Arial or Helvetica, and align it to the center of our website with this code: 


<style type="text/css">
	h1 {
	    text-align: center;
	    font-family: Arial Helvetica, sans-serif;
	}
</style>

The "Arial Helvetica, sans-serif" portion of the code denotes that the browser will first try to render the text in the Arial font, then if the user's computer doesn't have that font to try Helvetica, then any sans-serif font.
Now, we'll add some text to the site. In the body section of your file, add a <p> tag. Put a description of your website between the tags. We'll style it in the exact same way as our header.
We'll also change the background color and text color by adding a body section to our CSS and adding this to it: background-color: black; color: white; You can change the colors to whatever you want.
Our code should now look like this: 

<!DOCTYPE html>
<html>
	<head>
		<title>
			My Website
		<title>
		<style type="text/css">
		h1 {
			text-align: center;
			font-family: Arial Helvetica, sans-serif;
		}
		p {
			text-align: center;
			font-family: Arial Helvetica, sans-serif;
		}
		body {
			background-color: black;
			color: white;
		}
		</style>
	</head>
	<body>
		<p>
		But I must explain to you how all this mistaken idea of 
		denouncing of a pleasure and praising pain was born and
		I will give you a complete account of the system, and 
		expound the actual teachings of the great explorer of 
		the truth, the master-builder of human happiness. No 
		one rejects, dislikes, or avoids pleasure itself, because 
		it is pleasure, but because those who do not know how to 
		pursue pleasure rationally encounter consequences that 
		are extremely painful. Nor again is there anyone who loves 
		or pursues or desires to obtain pain of itself, because it
		is pain, but occasionally circumstances occur in which toil 
		and pain can procure him some great pleasure. To take a
		trivial example, which of us ever undertakes laborious 
		physical exercise, except to obtain some advantage from it? 
		But who has any right to find fault with a man who chooses
		to enjoy a pleasure that has no annoying consequences, or 
		one who avoids a pain that produces no resultant pleasure?
		</p>
	</body>
</html>
