CREATING A JAVASCRIPT GAME

In this tutorial, we'll create a basic HTML and JavaScript game.

Creating the HTML File

We'll create our game using the HTML canvas element. If you haven't already, I recommend reading my previous intro to HTML tutorial.
First, we'll create a basic HTML page with only a canvas element with a width and height attribute of whatever you want. Mine will be 500 x 300. We'll also add an "id" attribute to the canvas element, and set it to "canvas". Your code should look like this: 


<!DOCTYPE html>
<html>
	<head>
		<style type="text/css">
		</style>
	</head>
	<body>
		<canvas width="800" height="500" id="canvas">
		</canvas>
		<script type="text/javascript">
		</script>
	</body>
</html>

Now we'll start the new stuff. To create our game, we'll draw on the canvas element using JavaScript. First, though, we'll add some CSS. In the style element, make the canvas have these attributes: display: block; margin: 0 auto; border: 1px solid black; This will align the canvas to the center of your screen by displaying it as a box, and setting the margins to auto. It'll also create a 1 pixel wide solid black border, so that we can see the canvas.
We'll now start using JavaScript!
