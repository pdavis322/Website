INTRODUCTION TO HTML

In this tutorial, I'll explain the basics of the language and a bit of its interaction with JavaScript and CSS.

The Languages of the Internet

All Web pages consist of at least one HTML file. Think of HTML as a foreign language. Your computer can't understand this language, so your browser translates it into something your computer can understand. Browsers such as Google Chrome, Mozilla Firefox, and Microsoft Edge interpret the HTML and display it on a Web page.

Let's think of a Web page as a school. The HTML files are just the stuff in the school: the building, the desks, the textbooks. Students could theoretically learn with just textbooks, but the experience would be probably ineffective and certainly boring. That's why we have JavaScript.

Think of JavaScript as the teachers in the school. Teachers hopefully make learning fun and engaging. Similarly, JavaScript makes websites dynamic , giving visitors a more lively experience. While it is a programming language, JavaScript is much less strict than other traditional languages such as Java and C++, so don't be intimidated!

Back to our school metaphor, we now have both teachers and learning materials. But the school still lacks something without CSS: the posters, the art, the general decor. That's where CSS, or Cascading Style Sheets, comes in. CSS can alter a Web page's fonts, colors, alignment, and virtually anything else. CSS is essential in making a Web page look interesting and attractive to users.

In summary, HTML controls the content of a Web page, JavaScript controls what it does, and CSS controls what it looks like.
